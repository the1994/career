Get used to 'yum' and 'apt-get', so in Mac it can be a little bit wired without command line installation.

##Brew##

Install Brew:

    curl -LsSf http://github.com/mxcl/homebrew/tarball/master | sudo tar xvz -C /usr/local --strip 1

Use Brew:

    sudo brew install ...
    sudo brew update ...
